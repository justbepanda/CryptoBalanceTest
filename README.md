# Crypto Balance Test

Решение технического задания по управлению крипто-балансом пользователя на PHP + Laravel 12.

Позволяет корректно и безопасно:
- Зачислять средства
- Списывать средства
- Обрабатывать асинхронно события блокчейна
- Учитывать риски (double spend, race conditions)

---

## Быстрая установка

```bash
make init
```
